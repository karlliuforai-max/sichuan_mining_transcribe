# 项目背景：
我是一个矿业企业的负责人，常常会参与一些会议或商讨。我会想把这些会议用录音的方式记录下来，并且通过AI转写成文本进而生成会议纪要。
不过我现在遇到了一个问题，那就是由于我的工作区域主要集中在四川省（更集中地在乐山市），因此我的会议参与方大部分是四川乐山人，他们说着四川地区的不同方言，其中以成都方言和乐山方言居多。而目前很多转写软件，对于四川话，尤其是乐山话的识别正确率非常差劲（乐山话的口音跟普通的四川方言差距很大）。
因此我想做一个skill，这个skill的主要功能，是将我上传的由其他工具从录音直接转写过来的文本，进行四川话，尤其是乐山话的纠错，领域主要以矿业领域为主。
# 项目要求
1. 整个过程一定要连续，从我提供转写源到生成会议纪要，不需要我在中间操作，但是如果转写时间很长，例如1分钟以上，需要持续地向我提供进程信息，如已纠错xx字/共xx字等。以便我知道程序还在运行
2. 我提供的转写源可能是pdf的或者txt格式，请都要兼容
3. 我提供了一些转写源，这些都是真实的会议，里面可能夹杂着普通话、四川话、乐山话等
4. 我的会议领域主要是矿业领域，因此你可以针对矿业产业的相关用词做重点优化
